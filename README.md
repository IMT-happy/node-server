## 启动

```
npm install

node app.js
```

## 源码目录介绍

```
├── app                        // 主项目目录
│   ├── controllerts            // 接口服务入口
│   └── utils                  // 工具库
│   │   ├── formatDate.js      // 时间格式化
│   │   └── testcase.js        // 自动化case键值对配置
├── routes                     // 路由目录
│   └── api                    // 接口目录
│   │   ├── course_router.js   // 相关接口
├── app.js                     // 项目入口
└── config.js                  // 基础配置信息
```
